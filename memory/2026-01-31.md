- Configure ha_config.json with their HA URL and access token

## Usage Commands

Key commands available:
- node ha_tool.js test (test connection)
- node ha_tool.js entities (get all entities)
- node ha_tool.js entity <entity_id> (get specific entity)
- node ha_tool.js light <entity_id> [brightness] [color] [on/off]
- node ha_tool.js switch <entity_id> [on/off]
