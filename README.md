# BBS Monitor Chrome Extension

这是一个Chrome浏览器扩展，用于监控BBS论坛的新帖子，当检测到包含指定关键字的帖子时，会发送通知提醒您。

## 功能特性

- 监控指定的BBS论坛URL
- 检测包含关键字的新帖子
- 避免重复通知同一帖子
- 可自定义监控间隔
- 支持多个关键字和多个URL

## 安装步骤

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择此文件夹（bbs-monitor-extension）
6. 插件安装完成

## 配置说明

1. 点击浏览器右上角的扩展图标
2. 选择"BBS Monitor Options"进行配置
3. 在设置页面填写：
   - **监控的URL**: 要监控的BBS论坛地址（每行一个）
   - **监控关键字**: 要监控的关键字（用逗号分隔）
   - **检查间隔**: 检查频率（单位：秒）

## 使用方法

1. 配置完成后点击"保存设置"
2. 点击扩展图标可以启用/禁用监控
3. 当检测到包含关键字的新帖子时，会显示通知
4. 点击通知中的"打开帖子"按钮可直接访问该帖子

## 注意事项

- 为了能够访问外部网站，扩展已请求必要的权限
- 扩展会在后台定期检查指定的URL
- 所有数据均存储在本地，不会上传到任何服务器

## 技术细节

- 使用Manifest V3规范
- 内容脚本监控页面变化
- 后台脚本执行定时检查和通知
- 本地存储记录已通知的帖子，避免重复提醒
